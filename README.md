## General philosophy

Good tooling should make the operator’s life easier.

Not everything needs to be grandiose. Sometimes the best tool is just the thing that saves you from copying the same command, rewriting the same note, rebuilding the same redirector, or fixing the same report section for the tenth time.

I like tools that:

* reduce friction
* clean up workflows
* make tradecraft repeatable
* make reporting less painful
* help operators stay organized
